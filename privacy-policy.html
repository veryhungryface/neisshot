🏀 나이스샷 개인정보처리방침

최종 업데이트: 2026년 1월 6일

1. 개요

나이스샷(이하 "본 확장프로그램")은 사용자의 개인정보를 소중히 여기며, 개인정보보호법 등 관련 법령을 준수합니다.

2. 수집하는 개인정보

본 확장프로그램은 사용자의 개인정보를 수집하지 않습니다.

  • 사용자가 입력하는 데이터는 로컬에서만 처리되며, 외부 서버로 전송되지 않습니다.
  • 입력 속도 설정 등의 사용자 환경설정은 브라우저의 로컬 저장소에만 저장됩니다.

3. 권한 사용 목적

본 확장프로그램이 요청하는 권한의 사용 목적은 다음과 같습니다:

  • 호스트 권한 (neis.go.kr): 나이스 웹사이트의 입력 필드에 텍스트를 자동 입력하기 위함
  • activeTab: 현재 탭에서 입력 작업을 수행하기 위함
  • scripting: 입력 자동화 스크립트를 실행하기 위함
  • sidePanel: 사이드패널 인터페이스를 제공하기 위함
  • storage: 사용자 설정을 로컬에 저장하기 위함

4. 데이터 전송

본 확장프로그램은 어떠한 데이터도 외부 서버로 전송하지 않습니다. 모든 작업은 사용자의 브라우저 내에서만 이루어집니다.

5. 제3자 제공

본 확장프로그램은 사용자 데이터를 제3자에게 판매, 공유, 전송하지 않습니다.
